# 参考音频文件说明

此目录用于存放预设说话人的参考音频文件，用于零样本声音克隆。

**✨ 新增功能**: 系统会自动扫描此目录中的所有 `.wav` 文件，无需手动配置！

## 📁 文件命名规范

系统支持自动识别以下命名模式：

### 标准命名（推荐）

```
female_1.wav     -> 显示为: 女声1
female_2.wav     -> 显示为: 女声2
male_1.wav       -> 显示为: 男声1
male_2.wav       -> 显示为: 男声2
neutral_1.wav    -> 显示为: 中性1
child_1.wav      -> 显示为: 童声1
```

### 自定义命名

```
female_sweet.wav    -> 显示为: 女声sweet
male_deep.wav       -> 显示为: 男声deep
custom_voice.wav    -> 显示为: custom_voice（保持原样）
张三.wav            -> 显示为: 张三
```

## 📝 添加文本描述（可选）

为每个音频文件创建同名的 `.txt` 文件来提供参考文本：

```
prompt_audios/
├── female_1.wav
├── female_1.txt     # 对应的文本描述
├── male_1.wav
└── male_1.txt       # 对应的文本描述
```

**female_1.txt 示例内容**:
```
喜欢攀岩、徒步、滑雪的语言爱好者，以及过两天要带着全部家当去景德镇做陶瓷的白日梦想家。
```

如果没有 `.txt` 文件，系统会自动生成默认描述。

## 音频要求

- **格式**: WAV (PCM 16-bit)
- **采样率**: 22050 Hz 或 24000 Hz
- **时长**: 3-10 秒
- **内容**: 清晰的普通话语音
- **质量**: 无背景噪音，发音清晰

## 获取参考音频的方法

1. **录制**: 使用麦克风录制清晰的语音
2. **提取**: 从现有音频/视频中提取高质量片段
3. **下载**: 从公开语音数据集中选择合适的样本

## 方言支持

使用**普通话**参考音频即可，模型会自动进行跨方言克隆（Dialect-Guided Prompting）。
无需为每个方言单独准备参考音频。

## 示例文本建议

录制参考音频时可以使用以下文本：

```
"你好，欢迎来到 SoulX 播客。今天我们将讨论人工智能技术的最新进展。"
```

## 注意事项

- 参考音频的音色将直接影响生成语音的质量
- 建议选择语速适中、发音标准的音频
- 避免使用含有背景音乐或噪音的音频
