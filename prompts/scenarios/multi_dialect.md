# 多方言播客脚本生成提示词

> 专门用于生成包含多种方言的播客脚本，展现地域特色和文化多样性

---

## 任务说明

你的任务是将对话内容转换为多方言混合的播客脚本，为不同角色分配合适的方言，使播客更具地域特色和文化趣味。

---

## 快速提示词

```
请将以下对话转换为多方言播客脚本：

可用方言：
- 普通话：标准普通话，适合主持人、正式场合
- 四川话：四川方言，热情活泼
- 河南话：河南方言，朴实亲切
- 粤语：粤语/广东话，现代都市感

要求：
1. 根据角色背景分配方言
2. 主持人通常使用普通话
3. 嘉宾根据来源地使用相应方言
4. 保持对话内容的可理解性

对话内容：
[粘贴对话]
```

---

## 方言特点与适用场景

### 普通话

**特点**:
- 标准、正式、中性
- 易于理解，适合大众
- 专业、权威感强

**适用角色**:
- 主持人（主导对话）
- 专家、学者
- 新闻播报员
- 官方代表

**示例**:
```
[主持人]: 各位听众朋友大家好，欢迎收听今天的节目。
[专家]: 感谢邀请，很高兴有机会和大家分享。
```

### 四川话

**特点**:
- 热情、幽默、生动
- 语气轻松、亲切
- 富有生活气息

**适用角色**:
- 成都/四川来的嘉宾
- 讲故事的人
- 美食相关话题
- 轻松娱乐节目

**示例**:
```
[四川朋友]: 哈喽大家好，我来自成都。<|laughter|> 我们那边火锅是真的巴适得板！
[四川嘉宾]: 要得要得，这个事情我晓得。
```

### 河南话

**特点**:
- 朴实、真诚、接地气
- 语气实在、直接
- 民间文化气息浓厚

**适用角色**:
- 河南来的朋友
- 农村话题
- 传统文化讨论
- 家乡情怀

**示例**:
```
[河南朋友]: 中，这个我知道，俺们那边也是这样。
[河南嘉宾]: 恁说得对，就是这个理儿。
```

### 粤语

**特点**:
- 现代、时尚、国际化
- 节奏明快、活泼
- 商业、都市感强

**适用角色**:
- 广东/香港来的嘉宾
- 商业、科技话题
- 都市生活讨论
- 流行文化

**示例**:
```
[粤语朋友]: 大家好呀，我系广东人。我哋嗰度最出名嘅当然系早茶啦。
[粤语嘉宾]: 冇问题，呢个我好熟。
```

## 方言分配策略

### 策略 1：基于地域背景

根据角色的来源地分配方言：

```
# 家乡话题播客
@角色: 主持人, 女声1, 普通话
@角色: 成都朋友, 男声1, 四川话
@角色: 郑州朋友, 女声1, 河南话
@角色: 广州朋友, 男声1, 粤语

[主持人]: 今天我们请来了三位来自不同地方的朋友。
[成都朋友]: 哈喽，我从成都来。
[郑州朋友]: 俺是河南人。
[广州朋友]: 我系广州嚟嘅。
```

### 策略 2：基于话题内容

根据讨论的话题选择方言：

**美食话题 - 四川话**:
```
@角色: 主持人, 女声1, 普通话
@角色: 美食专家, 男声1, 四川话

[主持人]: 今天我们要聊四川美食。
[美食专家]: 巴适得很！<|laughter|> 我们四川的火锅那是相当安逸。
```

**商业话题 - 粤语**:
```
@角色: 主持人, 女声1, 普通话
@角色: 企业家, 男声1, 粤语

[主持人]: 请问您如何看待市场变化？
[企业家]: 我哋要抓住机会，积极应对。
```

### 策略 3：混合搭配

在同一对话中混合多种方言，增加趣味性：

```
@角色: 主持人, 女声1, 普通话
@角色: 川菜厨师, 男声1, 四川话
@角色: 粤菜厨师, 女声1, 粤语

[主持人]: 两位大厨，你们觉得川菜和粤菜有什么不同？
[川菜厨师]: 我们川菜讲究麻辣鲜香，巴适得很！<|laughter|>
[粤菜厨师]: 我哋粤菜就比较清淡，讲究原汁原味。
[主持人]: 听起来各有特色呢。<|laughter|>
```

## 完整示例

### 示例 1：地域文化交流

**主题**: 不同地方的过年习俗

```
# 角色定义
@角色: 主持人, 女声1, 普通话
@角色: 成都朋友, 男声1, 四川话
@角色: 广州朋友, 女声1, 粤语

# 对话内容
[主持人]: 大家好，今天我们聊聊各地的过年习俗。先请成都的朋友分享一下。
[成都朋友]: 好嘞！<|laughter|> 我们成都过年那是相当热闹，大家都要去锦里、宽窄巷子耍。
[主持人]: 听起来很有意思。那广州呢？
[广州朋友]: 我哋广州人过年就钟意去花市，行花街，好热闹架。<|laughter|>
[主持人]: 两个地方都很有特色。<|sigh|> 真想都去体验一下。
[成都朋友]: 欢迎欢迎，来了保证你巴适得不想走！
[广州朋友]: 我哋广州都随时欢迎你嚟玩。
```

### 示例 2：美食对话

**主题**: 火锅文化

```
# 角色定义
@角色: 主持人, 女声1, 普通话
@角色: 火锅店老板, 男声1, 四川话
@角色: 美食博主, 女声1, 普通话

# 对话内容
[主持人]: 今天我们要聊的是火锅文化。<|laughter|> 我们请到了一位资深的火锅店老板。
[火锅店老板]: 哈喽大家好！说起火锅，那我就来劲了。<|laughter|> 我们四川火锅那是真的巴适。
[美食博主]: 我也是火锅的忠实粉丝。请问正宗的四川火锅有什么讲究？
[火锅店老板]: 哎呀，这个讲究可多了。首先底料要地道，牛油要够香。<|sigh|> 还要有各种蘸料。
[主持人]: 听您这么一说，我都馋了。<|laughter|>
[火锅店老板]: 欢迎来我店头耍，保证让你吃得巴巴适适的！
```

### 示例 3：商业访谈

**主题**: 粤港澳大湾区发展

```
# 角色定义
@角色: 主持人, 女声1, 普通话
@角色: 企业家, 男声1, 粤语
@角色: 经济学者, 男声1, 普通话

# 对话内容

[主持人]: 企业家先生，您觉得最大的机遇在哪里？
[企业家]: 我觉得系科技创新同埋金融服务呢啲方面。<|breathing|> 我哋要把握机会。
[经济学者]: 我同意这个观点。<|sigh|> 创新是驱动发展的关键。
```

## 注意事项

### ✅ 应该做的

1. **保持可理解性**: 即使使用方言，也要保证大部分听众能理解
2. **符合角色背景**: 方言分配要符合角色的地域背景
3. **适度使用**: 不要过度使用方言特色词汇，影响理解
4. **主持人用普通话**: 通常主持人使用普通话，便于引导和衔接
5. **文化尊重**: 使用方言时要尊重当地文化，避免刻板印象

### ❌ 不应该做的

1. **不要生硬**: 不要为了用方言而用方言，要自然流畅
2. **不要歧视**: 不要用方言开玩笑或嘲讽
3. **不要过度**: 不要全篇都是方言，影响可懂度
4. **不要混乱**: 不要让同一角色在不同句子中切换方言
5. **不要误用**: 不要使用不熟悉的方言，可能造成错误

## 方言使用建议

### 轻度使用（推荐）

保留方言特色，但保持大部分内容的可理解性：

```
[四川朋友]: 欢迎大家来成都耍。<|laughter|> 我们这边好吃的东西巴适得很。
```

**特点**: 
- 主要内容用普通话
- 关键词使用方言（如"耍"、"巴适"）
- 易于理解

### 中度使用

增加方言比例，展现更多地域特色：

```
[四川朋友]: 你晓得不，我们成都的火锅那是真的巴适。要得很！<|laughter|>
```

**特点**:
- 方言词汇较多
- 保持基本可理解
- 地域特色明显

### 重度使用（谨慎）

大量使用方言，可能影响理解：

```
[四川朋友]: 你晓不晓得嘛，我们成都的火锅那是巴巴适适的。要得很噻！
```

**特点**:
- 方言比例很高
- 可能需要字幕
- 仅适合特定节目

## 质量检查

生成多方言脚本后，检查：

- [ ] 方言分配符合角色背景
- [ ] 内容整体可理解
- [ ] 方言使用适度，不影响体验
- [ ] 主持人使用普通话引导
- [ ] 文化表达尊重、得体
- [ ] 格式符合系统要求

---

## 使用建议

1. **了解方言特点**: 使用前了解各方言的特点和适用场景
2. **考虑受众**: 根据目标受众决定方言使用程度
3. **测试效果**: 生成后可以先测试一小段，看效果如何
4. **保持平衡**: 在地域特色和可理解性之间找到平衡
