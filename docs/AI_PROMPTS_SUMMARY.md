## 📖 使用示例

### 示例 1: 快速格式化对话

**输入 AI**:
```
[复制主提示词]

请格式化以下对话：
A: 大家好，欢迎来到今天的节目。
B: 你好，很高兴来到这里。
```

**AI 输出**:
```
# 角色定义
@角色: 主持人, 女声1, 普通话
@角色: 嘉宾, 男声1, 普通话

# 对话内容
[主持人]: 大家好，欢迎来到今天的节目。
[嘉宾]: 你好，很高兴来到这里。
```

### 示例 2: 添加情感标签

**输入 AI**:
```
[复制情感标签提示词]

请为以下脚本添加情感标签：
[主持人]: 这个问题很有意思。
[嘉宾]: 是的，我也这么认为。
```

**AI 输出**:
```
[主持人]: 这个问题很有意思。<|laughter|>
[嘉宾]: 是的，我也这么认为。
```
